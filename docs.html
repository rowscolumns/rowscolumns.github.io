Introduction

A robust, native-syntax two dimensional layout engine. Write real JavaScript logic directly in your HTML attributes without eval or Regex parsers.

Most grid systems force you to write endless nested <div>s. Rowscolumns is a recursive layout engine based on geometry. It supports React, Vue, and Vanilla HTML out of the box. 

<div layout="Grid.col(30, 70)">
    <div>Sidebar</div>
    <div>Content</div>
</div>

GETTING STARTED

Installation

Zero dependencies. Extremely lightweight (6KB).

1. NPM (Recommended)

npm install rowscolumns

2. CDN (Vanilla HTML)

<script type="module">
  import { Engine } from 'https://unpkg.com/rowscolumns';
  Engine.init();
</script>

CORE CONCEPTS

Basics: Columns & Rows

The library extends Number.prototype. Grid is a constant for 100. You can chain methods from it.

Making Columns

Use .col(a, b, c...) to split space horizontally.

<div layout="Grid.col(20, 80)">
  <div>20</div>
  <div>80</div>
</div>

Tip : You can make infinite columns Grid.col(10, 10, 10, 10, 10, 10, 10, 10, 10, 10...)

Making Rows

Use .row(a, b, c...) to split space vertically.

<div layout="Grid.row(30, 70)">

Making Grid (Recursion)

You can nest rows inside columns infinitely. Here is a 50/50 split, where the right side is split into 3 rows.

2. Variables (Golden Ratio)

The engine provides built-in variables: lg (61.8) and sm (38.2).

<div layout="Grid.col(lg, sm)">

3. Global Functions

You can define a layout function in your script and call it in HTML.

<script>
  window.myLayout = (gap) => Grid.col(50, 50).props({ gap });
</script>
<div layout="myLayout('20px')"> ... </div>

Controlling Dimensions

The number Grid (or 100) defines the ratio weights, not the actual pixel size. To change dimensions, apply CSS using .props().

<div layout="Grid.col(50, 50).props({ width: '50%', margin: '0 auto' })">
  <div>Left</div>
  <div>Right</div>
</div>

Result: A centered container taking up only 50% of the width.

Styling & Utilities

.props({ styles })

Apply CSS to the grid container (like gap, padding, margins).

Grid.col(50, 50).props({ gap: '20px', padding: '10px' })

.childProps({ styles }, [indices])

Apply CSS to specific children. Indices are 1-based.

Grid.col(30, 30, 30).childProps({ background: 'salmon' }, [2])

Creates "Ghost" slots. Useful for creating gaps. In this demo, we skip the middle slot (Index 2), forcing the 2nd Box to jump to the 3rd slot.

Golden Ratio

You can create a golden-ratio layout (Fibonacci spiral) using the built-in lg and sm variables (lg = 61.8, sm = 38.2).

Each split rotates the layout, and the smaller section becomes the container for the next split, producing a centered spiral:

<div layout="Grid.col(
  lg,
  sm.row(
    lg,
    sm.col(
      sm.row(
        sm.col(
          lg,
          sm.row(
            lg,
            sm.col(
              sm.row(
                sm.col(lg, sm)
              , lg)
            , lg)
          )
        , lg)
      , lg)
    , lg)
  )
)">

Complex Symmetry

Creating a symmetrical 23-element layout in standard HTML requires complex nesting or manual percentage calculations. With Rowscolumns, you just mirror the layout string.

Comparison: In standard CSS, you would need to calculate width: 4.34%, width: 8.69%, etc., or use complex Flexbox nesting. Here, it is one line of logic.

<div layout="Grid.col( 
  (10).row(1,1,1,1,1,1,1,1),  // Left Wing (8 rows)
  (20).row(1,1,1),            // Inner Left (3 rows)
  40,                         // Center (1 big col)
  (20).row(1,1,1),            // Inner Right (3 rows)
  (10).row(1,1,1,1,1,1,1,1)   // Right Wing (8 rows)
)">

Massive Rhythm (20 Columns)

Handling 20+ columns in native CSS Grid often involves tedious repeat() syntax or manual fractional math. Rowscolumns handles the math for any number of elements instantly.

This example generates 20 columns that progressively grow larger, creating a visual rhythm.

<div layout="Grid.col(1, 2, 3, 4, 5, 6, 7 ... 20)">

Asymmetrical Balance

Standard grids often feel rigid. By using a 6-column base and the .spread() method, elements can cross the center axis, creating dynamic, asymmetrical balance.

This layout uses .offset() to create negative space and .spread() to bridge columns.

<div layout="Grid.col(
  (1).row(1,1,1,3).spread(['r1'],[3,4]),
  (1).row(2,1,3).spread(['r1'],[1]).offset([2,3]),
  (1).row(2,2,1,1).offset([1]).spread(['r1'],[1]),
  (1).row(1,1,2,2).offset([3]).spread(['r1'],[3]),
  (1).row(3,1,2).offset([3]).spread(['r1'],[2,1]),
  (1).row(3,1,1,1).offset([1,2])
).props({gap:'5px'})">

React & TypeScript

Rowscolumns ships with a dedicated React wrapper and full TypeScript definitions.

1. The Setup

import React from 'react';
import { Layout, Grid, lg, sm } from 'rowscolumns/react';

2. Native Syntax (Recommended)

Pass real JavaScript objects to the layout prop. No quotes needed.

export const App = () => {
  const sidebar = 250; 
  
  return (
    <Layout layout={Grid.col(sidebar, 'auto')} >
       <div>Fixed Sidebar</div>
       <div>Fluid Content</div>
    </Layout>
  );
};
